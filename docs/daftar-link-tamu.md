# Daftar Link Tamu

Gunakan parameter `to` di akhir link undangan untuk menampilkan nama tamu secara dinamis.

```text
index.html?to=steven-wijaya-dan-ibu
index.html?to=dani-karel-dan-ibu
index.html?to=kris-halim-dan-ibu
index.html?to=ibu-jumiaty
index.html?to=aryani-dan-suami
index.html?to=crew-hollywood
index.html?to=crew-aneka-murah
```

Jika sudah di-hosting, ganti `index.html` dengan alamat website undangan.
